# Vision to Voice Object Detection System

## 🔹 Description
This project detects objects using YOLO and gives voice output for visually impaired people.

## 🔹 Technologies
- Python
- YOLO
- OpenCV
- Text-to-Speech

## 🔹 How to Run
1. Install required libraries
2. Run main.py
3. Use camera or image

## 🔹 Output
Detects objects and gives voice output
